    1. 其他的一些快捷键

      ctrl + 后退: 删除整个单词

      ctrl + L :全选当前行

      ctrl + shift + K: 删除当前整行

      shift + home / end: 移动光标到行首/行尾

      ctrl + shift + L 自动查找并修改当前单词

      ctrl + Alt + ↑ 或↓ 向上/下添加光标， 用于同时添加/删除相同的内容
